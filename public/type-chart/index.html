Pokemon GO Type Effectiveness Chart

Pokemon GO uses different damage multipliers than the main series games. There are no true immunities — what would deal 0x damage in the main games deals 0.391x in Pokemon GO.

Pokemon GO Damage Multipliers

  • Super effective: 1.6× damage (2× in main games)
  • Double super effective: 2.56× damage (4× in main games)
  • Not very effective: 0.625× damage (0.5× in main games)
  • Double resistant: 0.391× damage (0.25× in main games)
  • Immune (triple resistant): 0.244× damage (0× in main games)

Key Type Matchups for Raids

  • Ice vs Dragon/Flying: 2.56× (Rayquaza, Dragonite, Salamence)
  • Rock vs Fire/Flying: 2.56× (Moltres, Charizard, Ho-Oh)
  • Ground vs Fire/Poison: useful for Heatran (2.56×)
  • Fighting vs Normal/Ice/Rock/Dark/Steel: wide coverage
  • Fairy vs Dragon/Fighting/Dark: key defensive and offensive type

Types with No Effect in Pokemon GO

These deal 0.391× instead of 0× (still hits, just very reduced):

  • Normal → Ghost
  • Ghost → Normal
  • Electric → Ground
  • Fighting → Ghost
  • Poison → Steel
  • Ground → Flying
  • Psychic → Dark
  • Dragon → Fairy
