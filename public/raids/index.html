How Raid Counter Strings Work

Each search string uses Pokemon GO's move type filter syntax. @1rock finds Pokemon with a Rock-type fast move, @2rock finds Rock-type charged move, and @3rock finds a second charged Rock-type move. All terms are joined with commas (OR logic) so you get the widest selection of effective attackers.

Example: Mewtwo Counter String

Mewtwo is weak to Bug, Ghost, and Dark. The counter string is:

@1ghost,@2ghost,@3ghost,@1dark,@2dark,@3dark,@1bug,@2bug,@3bug
